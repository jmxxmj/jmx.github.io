python高级特性

2017.07.07 stylem  热度 ℃

由一个例子来开启python高级特性的学习吧~

构造一个列表 1，3，5，7，9，…,99
你会想到可以用一个循环来实现:(注意空格缩进） 

>>>L=[]      
>>>n=1    
>>>while n <= 99:    
   L.append(n)    
   n=n+2

切片

切片可以轻松的取一个list或tuple的部分元素
比如>>>L=[‘A’,’B’,’C’]
取前三个元素:
你可能会想到:>>>[L[0],L[1],L[2]]
切片方法：>>>L[0:3]
L[0:3]表示从索引0开始取，直到索引3为止，但是不包括索引3。
切片的其它表达形式
比如>>>L[:]就可以原样复制一个list
比如>>>L[::5]表示对一个list每5个取1个
比如对一个tuple L=(1,2,3,4,5)效果等同于对list的操作
比如字符串L=’ABCDEF’也可以看成是一种list。>>>L[:3]得到的结果是’ABC’
切片操作一定程度上可以减少很多循环的麻烦

迭代

什么是迭代？迭代就是可以通过for循环来遍历这个list或者tuple

对于一个dict (第一种情况:迭代的是key）

>>>d={'a': 1, 'b': 2, 'c': 3}    
>>>for key in d:    
    print(key)

输出结果 

a    
c    
b

对于一个dict (第二种情况:迭代的是value） 

>>>d={'a': 1, 'b': 2, 'c': 3}    
>>>for value in d.values():    
    print(value)

输出结果 

1    
2    
3

对于一个dict (第三种情况:迭代的是value） 

>>>d={'a': 1, 'b': 2, 'c': 3}    
>>>for k,v in d.items():    
    print(k,':',v)

输出结果 

a : 1    
b : 2    
c : 3

当然，字符串也可以进行迭代 

>>> for ch in 'ABC'    
        print(ch)

输出结果 

A    
B    
C

任何可迭代对象都可以作用于for循环，包括我们自定义的数据类型，只要符合迭代条件，就可以使用for循环

列表生成式

什么是列表生成式？列表生成式是Python内置的非常简单却强大的可以用来创建list的生成式

比如要生成一个list[1, 2, 3, 4, 5] 

>>>list(range(1,6))
[1, 2, 3, 4, 5]

比如要生成一个list[1, 4, 9, 16, 25,…,100 ] 

>>>[x*x for x in range(1.101)]    
[1, 4, 9, 16, 25,...,100 ]

列表生成式也可以用两个变量生成list 

>>> d = {'x': 'A', 'y': 'B', 'z': 'C' }    
>>> [k + '=' + v for k, v in d.items()]    
['y=B', 'x=A', 'z=C']

一个有趣的操作
把list中所有的字符串变成小写 

>>> L = ['Hello', 'World', 'IBM', 'Apple']    
>>> [s.lower() for s in L]    
['hello', 'world', 'ibm', 'apple']

通过列表生成式，可以快速生成一个list

生成器

什么是生成器？只要把一个列表生成式的[]改成（）就创建了一个生成器

比如 

>>> L = [x * x for x in range(10)]    
>>> L    
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]    
>>> g = (x * x for x in range(10))    
>>> g    
<generator object <genexpr> at 0x1022ef630>

那么我们要怎样操作才能得到生成器中的每一个元素呢？
使用next()函数 

>>> next(g)    
0    
>>> next(g)     
1    
>>> next(g)     
4     
...    
>>> next(g)    
81    
>>> next(g)     
Traceback (most recent call last):    
   File "<stdin>", line 1, in <module>    
StopIteration

由此我们知道，generator保存的是算法，每次调用next(g)，就计算出g的下一个元素的值，直到计算到最后一个元素，没有更多的元素时，抛出StopIteration的错误。

当然我们依然可以用迭代的方法来更方便的获得这些返回值 

>>> g = (x * x for x in range(10))   
>>> for n in g    
        print(n)    
0    
1    
...    
81

把一个函数改成下边的生成器

def odd():    
    print('step 1')   
    yield 1    
    print('step 2')   
     yield(3)

调用 

>>> o=odd()   
>>>next(o)    
step 1    
1    
>>>next(o)    
step 2    
2    
...

迭代器

判断一个对象是否是Iterable对象 

>>> from collections import Iterable    
>>> isinstance([], Iterable)    
True    
>>> isinstance({}, Iterable)    
True    
>>> isinstance('abc', Iterable)    
True    
>>> isinstance((x for x in range(10)), Iterable)     
True    
>>> isinstance(100, Iterable)    
False

判断一个对象是否是Iterator对象 

>>> from collections import Iterator    
>>> isinstance((x for x in range(10)), Iterator)    
True    
>>> isinstance([], Iterator)   
False     
>>> isinstance({}, Iterator)    
False    
>>> isinstance('abc', Iterator)    
False

凡是可作用于for循环的对象都是Iterable类型
凡是可作用于next()函数的对象都是Iterator类型，它们表示一个惰性计算的序列
